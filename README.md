# Cyclone Amphan Animation

Animated visualization of Cyclone Amphan (May 2020) using Python, Cartopy, Matplotlib, and particle-based wind field simulation.

## Features

- Cyclone-centered wind visualization
- Particle trail animation
- Dynamic cyclone eye movement
- Historical cyclone track
- MP4 and GIF export
- Bay of Bengal regional map

## Requirements

```bash
pip install cartopy matplotlib numpy pillow
```

For MP4 export:

```bash
sudo apt install ffmpeg
```

## Run

```bash
python amphan_cyclone.py
```

## Output

- `amphan_cyclone.mp4`
- `amphan_cyclone.gif`

## Study Region

Bay of Bengal and surrounding coastal regions including:

- India
- Bangladesh
- Andaman Sea

## Cyclone Track

The animation uses an interpolated cyclone track representing Cyclone Amphan's movement from 18–21 May 2020.
